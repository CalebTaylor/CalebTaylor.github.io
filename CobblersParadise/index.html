Cobbler's Paradise - 2020
Phaser 3 / JavaScript / Aseprite


Welcome to Cobbler's Paradise. A relaxing iddler where you make shoes in a mysterious shoe shop.

The short of it:

Marking my first official game shipment, my game development partner Remy Luisant and I worked as employees developing Cobbler’s Paradise. I handled all of the art, UI design, and UI programming, and Remy handled all of the music and gameplay / systems programming. The project lasted 4 months. It was built with Phaser 3 with some custom JavaScript plugins, and was designed to run cross-platform on mobile and desktop devices.

The long of it:

4 months isn’t a long time to make a game, especially a commercial product, and especially considering I was in University full-time as well. We needed a comprehensive schedule, numerous design sessions, and the ability to pivot at a moment's notice. This design cycle was absolutely insane to navigate, but we succeeded in delivering exactly what the client wanted.

UI(Switch 2nd place with art)

The client needed the game to work on mobile and desktop. While Phaser has some support for responsive interfaces and anchor points, overall I found it lacked the granularity we needed. We needed pixel-perfect and controllable scaling, anchor points, and real-time responsiveness (rotating a phone screen or resizing a desktop window), so this called for a custom JS plugin. I applied my web development/UI skills to create a bespoke UI library for Phaser, with which one can align sets of UI elements to 9 anchor points on the screen: Top row, middle row, bottom row, and left edge, middle, or right edge for each.
